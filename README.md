# [FastAPI](https://fastapi.tiangolo.com/)
Learning FastAPI

## Installation of pip
```sudo apt install python3-pip```

## FastAPI Setup
1. Create the virtual Environment (venv)
```
python3 -m venv venv
```
2. Activate the virtual Environment (venv)
```
source venv/bin/activate
```
3. Install FastAPI
```
pip install "fastapi[standard]"
```
